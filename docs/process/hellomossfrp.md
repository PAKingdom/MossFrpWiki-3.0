# 创建一个属于你的穿透码
穿透码在mossfrp属于一键配置的秘钥，请勿公开或泄露，如误发送至公开群组，建议[#刷新激活码](/bothelp/#更新穿透码)

# 挑选一个节点
发送 `#节点列表` 至 群聊/私聊 <br />
命令帮助: [#节点列表](/bothelp/#节点列表)

# 创建一个节点
发送 `#新建穿透码`
```
指令帮助：#新建穿透码 <节点> <带宽> <天数>
tip: 天数不得小于3
反之会提示：只能创建3-360天的穿透码！
```
