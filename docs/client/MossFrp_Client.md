```
映射类型：-

本地地址：
本地端口：
==========设置名称==========
请输入映射名称：

请不要输入中文/特殊符号/空格
避免frpc无法识别导致无法启动！

```

```

Frpc端口配置写入完成
请确认配置是否有误
如无误输入Y将生成文件
有误输入N重新输入
输入Back可以返回上一项设置
==========确认==========
```
